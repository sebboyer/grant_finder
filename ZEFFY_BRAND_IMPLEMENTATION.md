# Zeffy Brand Implementation - Grant Finder

## ✅ Brand Guidelines Applied

### Color Palette
All three Zeffy brand colors have been implemented:

1. **Periwinkle (#5555e7)**
   - Primary buttons and call-to-action elements
   - Text links and interactive elements
   - Primary gradients in header

2. **Sea foam green (#09cfaf)**
   - Accent color in gradients
   - Location tags and badges
   - Hover effects and decorative elements
   - Used in combination with Periwinkle for visual interest

3. **Midnight blue (#0f0e5b)**
   - Primary text color for maximum readability
   - Dark gradients for depth
   - High contrast elements

### Typography
- **Poppins font family** loaded from Google Fonts
- Font weights: 400 (regular), 500 (medium), 600 (semi-bold), 700 (bold)
- Applied consistently across all UI elements
- Appropriate for large text displays per brand guidelines

### Design Elements

#### Rounded Corners ✓
- All sharp corners removed per brand guidelines
- Border radius: 12px - 20px for various elements
- Buttons use 50px (fully rounded) for softness
- Cards use 20px for gentle, approachable feel

#### Wind-Inspired Organic Shapes ✓
- Decorative wave at bottom of page
- Curved wave accent in header
- Circular gradient accents on grant cards
- Smooth transitions and flowing animations

### UI Components

#### Header
- Gradient background: Periwinkle → Midnight blue
- White text for accessibility
- Curved wave decoration

#### Search Filters
- Clean, minimal input fields
- Rounded corners (12px)
- Focus states with Periwinkle highlight
- Autocomplete dropdown with brand colors

#### Buttons
- Primary: Gradient (Periwinkle → Midnight blue)
- Secondary: White with Periwinkle border
- Fully rounded (50px radius)
- Smooth hover animations with lift effect

#### Grant Cards
- Rounded corners (20px)
- Gradient amount text (Periwinkle → Sea foam green)
- Sea foam green accent glow on hover
- Location tags with gradient backgrounds
- Smooth hover animations

#### Pagination
- Rounded buttons (12px)
- Active state with brand gradient
- Hover effects with color transitions

## Design Philosophy

Following Zeffy's brand book:
- ✅ **Avoid sharp corners** - All elements use rounded borders
- ✅ **Organic shapes** - Wind-inspired waves and curves
- ✅ **Soft, approachable feel** - Gentle gradients and smooth transitions
- ✅ **Accessible colors** - High contrast text, proper color pairing
- ✅ **Modern minimalism** - Clean layout, ample white space

## Accessibility Considerations

- Periwinkle on white backgrounds for readable text
- Midnight blue for primary text (high contrast)
- Sea foam green used for accents, not primary text
- Proper focus states for keyboard navigation
- Sufficient padding and spacing

## Visual Hierarchy

1. **Header**: Bold gradient with brand colors
2. **Search section**: White card with clear inputs
3. **Grant cards**: Individual tiles with hover effects
4. **Interactive elements**: Clear visual feedback
